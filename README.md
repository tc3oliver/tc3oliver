## 👋 About Me

I’m Oliver, a Senior AI Engineer and System Architect based in Taiwan.

My work sits between AI, backend systems, and infrastructure.
I build systems that connect models, data, APIs, workflows, and deployment environments into something reliable enough to use outside of a demo.

I started from mobile and web development, moved into backend architecture and DevOps, and now focus on AI system integration, agent workflows, RAG systems, inference platforms, evaluation pipelines, and internal developer tooling.

I care about systems that are:

* practical enough to ship
* observable enough to debug
* structured enough to maintain
* safe enough to operate
* flexible enough to evolve

## 🧪 Featured Project

### Personal R&D Agent OS

A local-first, model-agnostic, privacy-aware agent runtime for personal research workflows.

The project turns Markdown research notes into a searchable, citation-grounded, traceable, and evaluation-driven R&D assistant.

Core capabilities:

* research QA over personal notes
* hybrid retrieval with citations
* privacy-aware model routing
* structured output validation
* JSONL trace logging
* runtime tool permission gate
* research digest, topic exploration, and synthesis workflows
* regression and adversarial evaluation gates

This project is not meant to be just another chatbot.
It is an experiment in building a small but serious agent runtime: one where retrieval, privacy, citations, tools, traces, and evaluation are treated as first-class engineering concerns.
